Ele não me deixa estudar!

Neste caso, antes, eu diminui a imagem utilizando o GIMP. De acordo com as aulas do Professor Guanabara, a diminuição da imagem antes torna o site menos pesado, porém seria possível diminuir seu tamanho utilizando width e height, conforme observa-se no trecho de código abaixo:


<code>
<img src="Cat.jpg"
alt="Cat" width="104"
height="142">
</code>

Há outra forma de definir tamanho utilizando CSS, qual seja: 

<code>
<img src="Cat.jpg"
style="width:104px;height:142px;">
</code>

Para inserir a cor em um parágrafo. Podemos usar o CSS inline, incorporado ou o externo. Abaixo segue um exemplo de css inline: 

<code>
<p style="color:blue;">This is a paragraph.</p>
</code>

Para formatar a fonte em um parágrafo, podemos usar p seguinte codigo: 

<code>
<p style="font-family:courier;">This is a paragraph.</p>
</code>

Para alinhar um parágrafo, podemos usar o seguinte codigo: 

<code>
<p style="text-align:center;">This is a paragraph.</p>
</code>

Para definir o tamanho do texto, podemos usar o seguinte codigo: 

<code>
<p style="font-size:50px;">This is a paragraph.</p>
</code>

Para definir uma cor de background, podemos usar o seguinte codigo: 

<code>
<p style="background-color:yellow;">This is a paragraph.</p>
</code>

É possível atribuir importancia a determinada palavra ao acrescentarmos a tag <strong> ao redor da palavra, conforme essa demonstra. A enfase em uma palavra é demonstrada ao acrescentarmos a tag <em> ao redor da palavra, conforme essa demonstra. Para deixar uma palavra marcada deve-se utilizar a tag <mark> ao redor da palavra, conforme essa demonstra.
